# Shell scripting exercises

ls - list of directory

cd - change directory

pwd - present working directory

cp - copy files from source to distination

mv - move files from source to distination

mkdir - make directories

rmdir - delete directories

touch - change file timestamp or create empty files

find - Search for files in a directory hierarchy

locate - find the files by name

tree - Display directories in a tree-like format

chmod - change file permisssion

chown - change file owner and group

chgrp - change the group ownership

stat - Display files or file system status
